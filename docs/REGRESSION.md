# Regression Checklist

## Pages to Verify
- Home / Landing
- Command Center (launch globe)
- Sky Events
- Tonights Mission
- Comets
- Solar Map
- Updates (Live / News / X)

## Expected Indicators
- Navigation works and routes load without blank screens.
- Command Center shows globe and launch/crew widgets without errors.
- Sky Events shows upcoming events and the banner renders.
- Comets list loads and detail page opens.
- Solar Map loads without console errors.
- Live Streams page renders embeds without crashing.
- No console errors related to `@starkid/core` imports.
