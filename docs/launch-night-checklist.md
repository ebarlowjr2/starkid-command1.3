# Launch Night Checklist

Internal operational document for major space events.

## Pre-Event (24-48 hours before)

- [ ] Verify YouTube live detection endpoint is working
- [ ] Confirm monitored channels are correct
- [ ] Confirm news feeds are updating
- [ ] Verify featured mission page loads quickly
- [ ] Test site on:
  - [ ] Desktop
  - [ ] Tablet
  - [ ] Mobile

## Day-Of Event

- [ ] Set featured experience (e.g., Artemis / Live page)
- [ ] Confirm live embed works
- [ ] Monitor error logs
- [ ] Avoid deploying code within 2 hours of event

## During Event

- [ ] Watch for YouTube API quota issues
- [ ] Refresh live detection status if needed
- [ ] Avoid UI changes unless critical

## Post-Event

- [ ] Remove "LIVE" emphasis
- [ ] Update mission status if applicable
- [ ] Note any issues encountered
- [ ] Capture screenshots for documentation
